# Real-Time Group Chat Application

A real-time group chat application built using HTML, CSS, JavaScript, Python (Flask), and Socket.IO.

This application allows users to:
- Create unique chat rooms
- Join chat rooms
- Invite users using one-time invite links
- Chat in real-time
- Track active users
- Use dark/light theme
- Embed the application using iFrame

---

# Features

- Real-time messaging
- Unique room generation
- One-time invite links
- Admin and participant roles
- Live active users tracking
- Join/Leave notifications
- Delete messages for all users
- Dark & Light theme
- Mobile-inspired UI
- iFrame support
- Responsive chat layout
